Executive Summary

Visual Studio Code (VS Code) is a lightweight, flexible code editing desktop application. It has support for basically every coding language, and includes many accessibility options including multiple built-in and downloadable color themes, zoom settings, customizable keyboard shortcuts, and tons of downloadable extensions
